## 核心计算规则

### 今日涨跌
```
今日涨跌 = 份额 × (今日净值 - 昨日净值)
```

### Tushare净值更新时间
- **隔日早上 8:20 后**
- 周一早上更新的是周五净值

### QDII基金
- Tushare净值有T+1延迟（已在数据中体现）
- 计算涨跌时用对应日期的净值

### 份额计算
```
买入份额 = 买入金额 / 确认日净值
```
- 普通基金: T+1确认
- QDII基金: T+2确认

## GridSeed V3.0 策略

### 阶段判断
```python
if grid_base_nav is None:
    phase = 'ACCUMULATION'  # 建仓阶段
else:
    phase = 'GRID'          # 网格阶段
```

### 建仓规则 (step 0→6)
| step | 名称 | 触发 | 操作 |
|:---:|:---|:---|:---|
| 0-3 | L1-L4 | 跌≥3% | +15% |
| 4-5 | L5-L6 | 跌≥5% | +30% |

### 网格规则
- 跌≥3%: 买入100元
- 涨≥10%: 卖出该批次
- 闲置10交易日: +100元

### 卖出规则
| 阶段 | 条件 | 操作 |
|:---|:---|:---|
| 建仓期 | 收益≥15% | 卖50%，进网格 |
| 网格期 | 批次收益≥10% | 卖出批次 |
